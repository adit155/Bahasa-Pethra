            <h2>3. Struktur & Aturan Dasar</h2>
            <ul>
                <li>Bahasa: SVO</li>
                <li>Kata dasar dianggap nomina/maim</li>
                <li>Sufiks <code>-ca</code> → fiza zhi (verba)</li>
                <li>Sufiks <code>-tsa</code> → fiza sif (adjektiva)</li>
                <li>Sufiks <code>-kak</code> → agen</li>
                <li>Pronomina terikat: yoa</li>
                <li>Peralihan verba intransitif → transitif (mesatsa zhi → moisatsa zhi) hanya untuk verba tertentu</li>
                <li>Pentingnya logika: mog</li>
            </ul>
        </section>

        <section>
            <h2>4. Radikal & Turunan</h2>
            <h3>Radikal 1 (ı)</h3>
            <ul>
                <li>i → /j/, ï → /l/, ī → /r/</li>
            </ul>

            <h3>Radikal 2 (o)</h3>
            <ul>
                <li>¤ → /m/, ö → /ɲ/, ō → /ŋ/</li>
            </ul>

            <h3>Radikal 3 (ʃ)</h3>
            <ul>
                <li>ʃ̀ → /f/, ʃ̂ → /p/, ʃ́ → /v/, ʃ̌ → /w/</li>
            </ul>

            <h3>Radikal 4 (ɯ)</h3>
            <ul>
                <li>ɯ̈ → /خ/, ɯ̄ → /غ/, ɯ̆ → /ح/, ɯ̂ → /ع/</li>
            </ul>

            <h3>Radikal 5 (u)</h3>
            <ul>
                <li>∪ → /kʷ/, ū → /g/, ü → /q/, ⊔ → /ks/</li>
            </ul>

            <h3>Radikal 6 (c)</h3>
            <ul>
                <li>⊂ → /sʷ/, ⊏ → /ʃ/, č → /z/, ç → /dʒ/</li>
            </ul>

            <h3>Radikal 7 (n)</h3>
            <ul>
                <li>ń → /ʝ/, ń → /ɟ/, ņ → /t/, ṇ → /tʷ/, ṉ → /ts/, ṅ → /θ/, ŋ → /d/, ɳ → /dʷ/, ɲ → /ð/</li>
            </ul>
        </section>
